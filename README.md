## Stream Log

Stream Log is a lightweight and extensible logger library for Kotlin.

### Gradle Setup

Add the dependency below into your **module**'s `build.gradle` file:

```gradle
dependencies {
    implementation("io.getstream:stream-log:$versions")
}
```
